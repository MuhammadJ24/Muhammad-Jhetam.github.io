Football 21-22 Season Analysis
in SQL and Tableau

In this project data on the 21-22 Football season of Europe's top 5 leagues was analysed in SQL.
The insights generated from this analysis were then visualized in Tableau through the use of a dashboard.

Introduction

The analysis done in this project focused on stats of both players and teams in Europe's Top 5 football leagues over the course of the 21-22 season. This includes the Premier League of England, La Liga of Spain, The Bundesliga of Germany, Serie A of Italy and Ligue 1 of France.
The datasets used were obtained on Kaggle.com and can be found using the following links: Football Team Stats 21-22, Football Player Stats 21-22.

The SQL code base for the entire project can also be found in my Github Project Repository.
The dashboards for this project can be found here: Football 21-22 Dashboard 1, Football 21-22 Dashboard 2.


Data Preparation

The datasets used in this project were formatted in Microsoft Excel.

Data Formatting

All entries containing numerical values had their data type changed to a numerical data type , similarly the data type of all cells containing non-numerical data was changed accordingly.
All periods '.' were changed to commas ',' in the decimal part of numerical entries. This was done to correct a error, that was faced in SQL Server, where some numerical values containing periods '.' as a decimal point were displayed as Nulls. Changing the decimal points into commas ',' fixed this problem.

Data Inspection

The database was checked for Null entries, there were none.


Data Analysis

The data analysis in this project was done using SQL queries in SQL Server. Queries to the following were carried out:
  • Find the top scorer/scorers of each of the top 5 leagues.

  • Players that played the most minutes in the top 5 leagues per position (Foward, midfield, defense and goalkeeper).

  • Compare important stats of top three teams in each league such as points earned, goals scored, matches played, points per game, etc.

  • Finding the winning team of each of the top 5 leagues and their key players.


Data Visualization

Using the above queries , the data was visualized to create a brief summary of my findings in this project. using Tableau to create two interactive dashboards, the first sumarizing key takeaways from the 21-22 football season across europe's top 5 leagues. The second dashboard contains a more in-depth look at the important stats of the top 3 teams in these leagues.

Final Remarks

The visualizations created in this project give us valubale insight on the best performing teams and their key players in the considered leagues. For instance, the top scorer in each league tends to be playing for the team winning the league as seen with 4/5 of the leagues.
Another valuabe insight is that of the outfield players that played the most minutes across all the leagues, these players were from the 2 'smallest' leagues in consideration namely, Serie A(Italy) and Ligue 1(France), and the teams they were from included Cagliari(Serie A), Lens(Ligue 1) and Lille(Ligue 1) all of which a cosidered to be mid-low ranked teams in their specific leagues. Outfield players from these teams playing the number of minutes that they have could be an indication of a lack of squad depth in those positions.
