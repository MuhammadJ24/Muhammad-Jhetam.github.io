Covid Data Exploration
in SQL and Tableau

In this project data exploration on the Covid-19 pandemic was conducted in SQL.
Insights generated from this exploration were then visualized in Tableau through the use of a dashboard.

Introduction

The exploration done in this project was inpired by Data Analyst and Youtube creator, AlexTheAnalyst.
The dataset that was used can be found using the following link: CoronaVirus (Covid-19) Info.

The SQL code base for the entire project can also be found in my Github Project Repository.
The dashboard for this project can be found here: Tableau Public Covid Dashboard.


Data Preparation

Before beginning the data exploration phase of this project, the dataset in consideration had to be prepared to make it simpler to work with.

Importing Data

The first step in the data preparation process after downloading the excel file containing the data, was to split the data in the file into two separate excel files. The first file contains information on covid infection rates and deaths, the second, containing information on covid vaccinations, along with some other information.

Database Restructuring

Columns that were more 'relevant' in this project where moved to keep all important fields close together, making it easier to observe the data we'd like to observe. This is not necessary, however, I prefer this structure.


Data Exploration

The data exploration in this project was done using SQL queries in SQL Server. Queries to the following were carried out:
  • Total Cases vs. Total Deaths internationally and in South Africa.

  • Percentage of population infected per country.

  • Countries with highest infection rate.

  • Countries with highest death count.

  • Continent with highest death count.

  • Overall global death rate.

  • Percentage of vaccinated people in a population per country(Done in 2 different ways, first using a CTE and the other making use of a temp table.).

Data Visualization

Using the above queries (slighlty tweaked at times), the wanted data was visualized to create a brief summary of my findings in this project. using Tableau to create an interactive dashboard as seen below:


Covid Data Exploration Dashboard


Final Remarks

This project yielded some interesting insights visualized in the above dashboard, which include the average percentage of poeple infected over the course of the two year pandemic in the BRICS nations, the total number of deaths per continent and lastly, the percentage of infected people in each country at the height of the pandemic.
