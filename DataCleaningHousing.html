Data Cleaning


Date Reformatting

The first step in this project was to reformat the sale date of the properties in the dataset. Initially the sale date also included the time of the sale down to the milli-second! For example, 2013-04-09 00:00:00.000. It's also worth noting that none of these times had any values assigned to them. I reformatted the sale date to be more usable and sensible. Separating columns

Some columns contained valuable information, however, this information would have been hard to make use of because of the column's structure. Examples of this inlude: The Property Address column containing both the address and city of the property in a single column. Ideally we would have these in separate columns. Similarly, the Owner Address column contained the Address, City and State of residence of each owner in a single column. I seperated each of these into seperate columns for better usability.

Standardization of Data

The column indicating whether or not a property was vacant when it was sold had some inconsistencies. The entries Y and Yes both indicate that the property was indeed vacant, while the entries N and No both indicate that the property was occupied rather than vacant. I standardized these values to only indicate either a Yes or No value for consistency.
